## ğŸ¨ PRODUCTION NOTES

### Visual Style Guide
- **Color Palette**: 
  - Primary: Indigo (#4F46E5)
  - Secondary: Blue (#3B82F6)
  - Accent: Slate (#64748B)
  
- **Typography**:
  - Headings: Inter Bold
  - Body: Inter Regular
  - Code: JetBrains Mono

- **Animations**:
  - Smooth transitions (0.3s ease)
  - Gentle glow effects for DARA logo
  - Loading spinners for processing

### Audio Requirements
- **Voiceover**: Clear, friendly, professional tone
- **Background Music**: 
  - Uplifting corporate/tech style
  - 60-80 BPM
  - Volume: -20dB (under narration)
  
- **Sound Effects** (optional):
  - Button clicks (subtle)
  - Upload "whoosh" sound
  - Success "ping" when results appear

### Recording Setup
- **Screen Resolution**: 1920x1080 minimum
- **Frame Rate**: 30fps or 60fps
- **Browser**: Chrome (for consistent rendering)
- **Cursor**: Make visible and enlarged for clarity
